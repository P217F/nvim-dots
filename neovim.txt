**Notes: Config dotfiles at https://github.com/P217F/neovim-for-work.git**

- Keymaps set in normal mode:
F1: open telescope find_files
F2: open telescope live_grep
F3: open telescope oldfiles
F4: open nvimtree
F5: split
F6: vertical split
F7: open inc-rename
F8: open aerial
F9: open nvim-dap
F10: step over
F11: step in
F12: step out
<C-Left>, <C-Right>, <C-Up>, <C-Down>: resize buffer(when split)
<C-h>, <C-j>, <C-k>, <C-l>: change buffer(when split)
<C-z>, <C-y>: undo, redo
;t: open terminal
<M-m>: comment with lsp
<S-f>: format code
<M-a>, <M-d>, <M-h>, <M-n>, <M-p>: add_files_mark, remove_files_mark, toggle_quick_menu, next_files_mark, previous_files_mark
<leader>b, <leader>t: set and remove breakpoints, terminate nvim-dap
<leader>: show keymap list
<leader>pp: open telescope of project
<leader>pc: set current directory to root project
<leader>g: Preview hunk inline
gn,gp: next hunk, previous hunk
gd: Go to definition
K: Show mini windows of definition information
]f: jump to head of next function
]c: jump to head of next class
[f: jump to head of previous function
[c: jump to head of previous class
>a: swap with next parameter
<a: swap with previous parameter

- Keymaps set in visual mode:
af: select all function
if: select inner function
ac: select all class
ic: select inner class
ab: select all block
ib: select inner block

- Keymap default by vim/neovim(normal mode)
daw: delete a word in pointer(delete space)
di(: delete all in (*)
w: move to next word
e: move to end of word
O: Push present line to down and create new line(not delete)
s: delete an char in pointer and go to insert mode
S: delete all in line and go to insert mode
dd: delete a line
<Shift> + > + >: indent line(can use in visual mode to indent many line)
<Shift> + < + <: unindent line(can use in visual mode to unindent many line)
gh: go to select mode
v: go to visual mode
V: go to visual mode, wrap all in line
a: insert mode after char
A: insert mode at end of line
i: insert mode
I: insert mode in begin of line
/: go to find mode
n/N: go to next/previous word that you highlight in find mode
x: delete a char
dw: delete word after pointer(ignore space)
d$: delete to end of line
cw: change word
ciw: change word
c$: edit to end of line
cc: edit all line
yy: yank line
yiw: copy word
y$: copy to end of line
p/P: paste to after/before line

- Keymap default by vim/neovim(visual mode)
<C-v>: block
gv: reselect line that select before
